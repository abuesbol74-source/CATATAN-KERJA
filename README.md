## 🐉 1. CARA INSTALL DI KALI LINUX (Desktop/Ubuntu)
Salin perintah ini satu per satu:
```bash
sudo apt update && sudo apt install git bash nano -y
git clone https://github.com/abuesbol74-source/CATATAN-KERJA.git
cd CATATAN-KERJA
chmod +x kong.sh
./kong.sh
```
