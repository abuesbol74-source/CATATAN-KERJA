# CATATAN-KERJA
Alat pencatat log kerja harian (April 2026) via Terminal. Created by LAKSAMANA DZU NUR AIN.

# 🛠️ KONG KERJA - April 2026
> **Created by LAKSAMANA DZU NUR NAIN**

Script Bash sederhana untuk mencatat log jam kerja harian secara otomatis dengan tampilan terminal yang elegan.

---

## 🚀 Fitur Utama
* **Kalender Otomatis:** Menentukan hari (Senin-Minggu) secara otomatis berdasarkan tanggal di bulan April 2026.
* **Database Text:** Menyimpan semua input ke dalam file `catatan_kerja_april_2026.txt`.
* **Interface:** Tampilan Header ASCII yang tebal dengan warna high-contrast.

---

## 📥 Cara Instalasi & Menjalankan

### **1. Di Termux (Android)**
Buka Termux dan jalankan perintah berikut:
```bash
pkg update && pkg upgrade
pkg install git bash nano -y
git clone https://github.com/abuesbol74/CATATAN-KERJA.git
cd CATATAN-KERJA
chmod +x kong.sh
./kong.sh
